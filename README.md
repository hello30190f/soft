## 概要
　投票システムを作る。

## 実行方法
```
git clone git@github.com:hello30190f/soft.git
cd soft
python main.py
```

## ファイル構成
### main.py
　初めに実行される処理が記述されている。C言語でいうmain関数に相当する機能を持つ。

### data.py
　立候補者、投票のデータを管理するクラスが宣言されている。

### err.py
　ファイルのフォーマットに問題がないかなどを調べる関数群がある。

### process.py
　実際に当選者を見つける処理を行う。
